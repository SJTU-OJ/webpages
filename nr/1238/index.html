    1238 欢总买苹果

    题目

    Description

    这一天，欢总来到超市买苹果。欢总认为越大的苹果就会越好吃，所以欢总希望尽快挑出一些大苹果。虽然欢总只有两只手，但是欢总可以一次从面前拿起三个苹果（我是不会轻易告诉你们原因的！）。现在请你来帮助欢总从眼前的三个苹果中选出一个最大的。

    Input Format

    标准输入包含3行：

    第1行是整数A，表示第一个苹果的质量（g）。

    第2行是整数B，表示第二个苹果的质量（g）。

    第3行是整数C，表示第三个苹果的质量（g）。

    Output Format

    标准输出包含1行：

    第1行输出整数D，表示最大的苹果的质量（g）。

    Sample Input1 

    100
200
300

    Sample Output1 

    300

    Sample Input2 

    100
300
200

    Sample Output2 

    300

    Limits 

    100%的数据满足：1<=A, B, C<=200000
