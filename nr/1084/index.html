    1084 二哥的宠物

    题目

    Description

    二哥养了很多宠物，而且宠物们的关系很好，每个宠物都可以通过一系列它直接认识的或者间接认识的宠物，来认识任意一个其他的宠物。也就是说，所有宠物是连通的，不存在孤立的一个或几个宠物。

    二哥时常担心宠物们联合起来对付他，但是他听不懂宠物的语言，所以他想知道这些宠物最多有多少种互相认识的可能。

    比如，如果二哥有三只宠物，那么一共有四种可能：

    Input Format

    输入包含多行，每行为一个整数 n，表示二哥拥有的宠物的个数,满足 \(1 \leq n \leq 100\)。

    输入的末尾会有一个整数 0，表示输入的结束。

    Output Format

    对输入的每一行中的 n，输出在这一情况下的结果。

    Hint

    此题必须在运行时刻得出结果，选手程序会经过人工评判后计分。
