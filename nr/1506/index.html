    1506 猪仔的减肥计划

    题目

    Description

    小猪Mcdull为了赢得同学阿May的芳心，开始了自己的减肥计划。他在一条山路上开始了他的训练。Mcdull希望能在每次训练中跑得尽可能远，不过他也知道农场中的一条规定： 每只猪仔独自进山的时间不得超过 M 秒( 1 <= M <= 10,000,000 )。

    整条山路被小猪Mcdull划分成T个长度相同的小段( 1 <= T <= 100,000 )，并且，Mcdull用 S_i 表示第 i 个小段的路况。 S_i 为 u，f，d 这3个字母之一，它们分别表示 第i个小段是上坡、平地，或是下坡。

    Mcdull要花 U 秒( 1 <= U <= 100 )才能跑完一段上坡路，跑完一段平地的耗时是 F 秒( 1 <= F <= 100 )，跑完一段下坡路要花 D 秒( 1 <= D <= 100 )。注意，沿山路 原路返回的时候，原本是上坡路的路段变成了下坡路，原本是下坡路的路段变成了上坡路。

    小猪Mcdull想知道，在能按时返回农场的前提下，他最多能在这条山路上跑多远。你能帮助到这只可爱的猪仔嘛？

    Input Format

    • 第1行: 5个用空格隔开的整数： M，T，U，F ，以及 D
    • 第2..T+1行: 第i+1行为1个字母 S_i ，描述了第i段山路的路况

    Output Format

    • 第1行: 输出1个整数，为贝茜在按时回到农场的前提下，最多能跑到多远

    Sample Input 

    13 5 3 2 1
u
f
u
d
f

    Sample Output 

    3
